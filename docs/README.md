# Sportifka GitHub Pages

This folder contains the static pages needed for Google Play and store support links:

- `index.html` - support landing page
- `privacy.html` - privacy policy
- `delete-account.html` - account deletion request page
- `support.html` - support contact page

Contact details already set:

- Support email: `dandroinka@gmail.com`
- Developer: `dandroinka`

Recommended Google Play URLs after GitHub Pages is enabled for a project repository:

- Privacy Policy: `https://dandroinka.github.io/YOUR_REPOSITORY/privacy.html`
- Account Deletion: `https://dandroinka.github.io/YOUR_REPOSITORY/delete-account.html`
- Support: `https://dandroinka.github.io/YOUR_REPOSITORY/support.html`

If the repository is named `dandroinka.github.io`, the URLs are shorter:

- `https://dandroinka.github.io/privacy.html`
- `https://dandroinka.github.io/delete-account.html`
- `https://dandroinka.github.io/support.html`
