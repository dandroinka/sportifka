Account Deletion

This page explains how users can request deletion of a Sportifka account and related cloud data. Sportifka account access is optional and uses Google Sign-In.

Use the contact email below to request deletion.

How to request deletion

  1. Send an email to dandroinka@gmail.com with the subject line Sportifka account deletion.
  2. Include the Google email address used to sign in to Sportifka.
  3. Send the request from the same email address when possible. We may ask for additional confirmation to protect the account from unauthorized deletion.

Data deleted after a verified request

  • Firebase Authentication account associated with Sportifka, when identifiable.
  • Cloud workout history stored for the account.
  • Cloud profile and body data stored for the account (height, weight, age, goals, and the day-by-day weight history).
  • Cloud achievement records stored for the account.
  • Friend invite and friend request records associated with the account.
  • Feedback records associated with the account, where they can be identified.

Data not deleted automatically

  • Local data stored only on the user's device. To remove local data, uninstall Sportifka or clear the app's storage in Android settings.
  • Data stored in Health Connect. The user can remove Health Connect data using the Health Connect app or Android system settings.
  • Crash logs and diagnostic records that may be retained for a limited period for security, reliability, legal compliance, or abuse prevention.
  • Fully anonymized or aggregated records that can no longer identify the user.

Processing time

Verified deletion requests are processed within 30 days unless a longer period is required by law, security, or fraud-prevention obligations.
